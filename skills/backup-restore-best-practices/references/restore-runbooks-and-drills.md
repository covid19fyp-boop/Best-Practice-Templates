# Restore Runbooks And Drills

## Runbook Minimum Contents

- Preconditions and required approvals.
- Exact restore commands and parameter templates.
- Verification checkpoints and rollback/abort conditions.
- Incident ownership and communication plan.

## Drill Program

- Run restore drills on a fixed cadence.
- Include full restore and targeted object-level restore scenarios.
- Measure actual RTO/RPO against target objectives.

## Post-Restore Verification

- Validate schema migration state.
- Validate critical API and frontend smoke paths.
- Validate data correctness for high-risk entities.
- Record findings and feed improvements back into runbooks.
